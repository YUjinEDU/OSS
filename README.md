# CLI Rubik's Cube

파이썬으로 구현된 CLI 루빅스 큐브 시뮬레이터입니다.

## 실행 방법

```bash
python cube.py
```

## 조작법

- **화살표 키**: 큐브 시점 회전
- **U, D, L, R, F, B**: 각 면 회전 (Shift를 누르면 반대 방향)
  - U: Up (윗면)
  - D: Down (아랫면)
  - L: Left (왼쪽면)
  - R: Right (오른쪽면)
  - F: Front (앞면)
  - B: Back (뒷면)
- **Space**: 큐브 섞기
- **Q**: 종료
